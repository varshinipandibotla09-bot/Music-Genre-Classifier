# 🎵 Music Genre Classifier

## 📌 Project Overview
The Music Genre Classifier is a simple web application that predicts a music genre based on the selected tempo and energy level using JavaScript.

## 🚀 Features
- Predicts music genre
- Easy-to-use interface
- Instant results
- Responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
```
music-genre-classifier/
│── index.html
│── style.css
│── script.js
│── README.md
```

## ▶️ How to Run
1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Select the tempo and energy level.
4. Click **Predict Genre** to view the result.
